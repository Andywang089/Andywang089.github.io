供应商筛选与引入

发表于 2018-04-10 | 分类于 采购工作相关

供应商的筛选

  • 所有筛选过的供应商都要留存记录，做到可追溯。包括沟通记录，联系方式、如何确定入围，如何排除等等都要有记录，不仅仅是项目总结使用，在验收检查时候也能用到。

供应商的引入

  • 根据经营范围，天眼查等信息查到的结果做初步筛选，初步筛选无法满足的，省去后面的二次筛选。
  • 进入二次筛选的，着重关注年产量，销售量等情况。

选择高质量的供应商是供应链管理的关键

关于采购协议整合

发表于 2018-03-08 | 分类于 采购工作相关

关于采购工作的整合

背景

集团下属各公司独立设置采购业务单元，在从事采购业务时，各自独立操作，资源共享方面存在欠缺，且工作标准、协同作战方面也有待整合提高，基于公司转型大背景，奔着节约高效，资源共享目的，统一成立大的采购部，化零为整。

存在的个别问题

分公司的采购框架协议和总部框架协议进行整合，目前碰到一个案例，总公司的采购供应商姑且称为A，某分公司采购相同产品协议供应商为B。B供应商在分公司所在地在价格、属地配送方面存在一定优势，如果全部整合至一份框架协议中，停掉属地有优势的供应商B，这必然会一定程度的提升采购成本，财务归口也将提出异议。所以说现在的整合是什么样的整合精神，是一定要砍掉分公司的供应商，只保留总部大协议供应商，还是进一步考核，保留两个协议进行有选择的整合。

升级自己的操作系统

发表于 2018-02-12 | 分类于 李笑来《财富自由之路》读书笔记

读李笑来财富自由之路-升级操作系统章节有感

这一章节说到了升级自己的操作系统，我这里直接说观点，我们的注意力>时间>金钱，这个关系式直接和个人的价值观系统相联系，看上去简单但对自己的感触确实很大。三者关系顺序不同，价值观就不同，进而的选择就不同，选择不同结果自然不同。

  • 觉得时间最不值钱的例子：为了省十几块、几十块钱不舍得花钱打车，宁愿等公交车，或者骑共享单车回家。甚至更悲催的是时间也浪费了，公交车却迟迟不来，最后又不得已打车回家。
  • 注意力一直被免费收割的例子：葛优瘫在沙发上，刷着微博，朋友圈，看着明星，网红的表演，无关痛痒的娱乐信息等等，自己的时间，自己的注意力被悄悄的消费掉。没有产生一点价值。
